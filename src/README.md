# Source Code

Place your main source code here.

This directory should contain the core implementation of your project.

## Structure

Organize your code in a way that makes sense for your language and project:

- **Flat structure**: For simple projects, place all source files directly here
- **Modular structure**: For larger projects, create subdirectories by feature or component

## Example structures:

### Python
```
src/
├── __init__.py
├── main.py
├── models/
├── services/
└── utils/
```

### Node.js
```
src/
├── index.js
├── controllers/
├── models/
├── routes/
└── middleware/
```

### Go
```
src/
├── main.go
├── handlers/
├── models/
└── services/
```

### Java
```
src/
└── main/
    └── java/
        └── com/
            └── yourproject/
```
